数组模拟队列

发表于 2020-07-28

1.数组模拟队列

实现思路：
1.定义两个变量front（队列的最前元素（不包含））、rear（队列的最后的元素（包含））；
2.将尾指针rear往后移动：rear + 1，front == rear时，队列为空；
3.尾指针小于队列的最大下标maxsize - 1，可填入数据；
4.rear == maxsize - 1，队列满。

class ArrayQueue{
    private int maxsize;//数组的最大容量
    private int front;//队列的头部
    private int rear;//队列的尾部
    private int[] arr;//存放数据

    public ArrayQueue(int maxsize){
        this.maxsize = maxsize;
        arr = new int[this.maxsize];
        front = -1;//队列的第一个数据的前一个位置
        rear = -1;//队列尾部的数据；
    }

    //队列是否为满
    public boolean isFull(){
        return rear == maxsize - 1;
    }
    //判断队列为空
    public boolean isEmpty(){
        return rear == front;
    }

    //添加数据
    public void addQueue(int n){
        if (isFull()){
            System.out.println("队列满！");
            return;
        }
        rear++;
        arr[rear] = n;
    }

    //获取队列的数据
    public int getQueue(){
        if (isEmpty()){
            throw new RuntimeException("对列为空！");
        }
        front++;
        return arr[front];
    }

    //所有数据
    public void showQueue(){
        if (isEmpty()){
            System.out.println("对列为空！");
            return;
        }
        for (int i = 0;i < arr.length;i++){
            System.out.println(arr[i]);
        }

    }
}

2.数组模拟环形队列

实现思路：环形队列
1.front值为0；
2.rear值为0；
3.front == rear，队列为空；
4.（rear + 1） % maxsize == front；队列满
5.此时有效的数据个数为（rear + maxSize -front） % maxSize，用于遍历队列中的值.
class CircleArrayQueueDemo{
    int front = 0;
    int rear = 0;
    int maxSize;
    int[] ints;

    public CircleArrayQueueDemo(int maxSize){
        this.maxSize = maxSize;
        ints = new int[this.maxSize];
    }

    public boolean isFull(){
        return (rear + 1) % maxSize == front;
    }

    public boolean isEmpty(){
        return rear == front;
    }

    public void add(int n){
        if (isFull()){
            System.out.println("队列满！");
            return;
        }
        ints[rear] = n;
        rear = (rear + 1) % maxSize;//防止角标越界
    }

    public int getQueue(){
        if (isEmpty()){
            System.out.println("对列为空！");
        }
        int value = ints[front];
        front = (front + 1) % maxSize;
        return value;
    }

    public void showArrayQueue(){
        if (isEmpty()){
            System.out.println("对列为空！");
        }
        for (int i = front;i < size();i++){
            System.out.println(ints[i % maxSize]);
        }
    }

    public int size(){
        return (rear + maxSize -front);//获取有效的数据个数
    }
}

1、快速排序

public class Test01 {
    public static void main(String[] args) {
        int[] ints = {1, 2, -8, -9, 52, -9, 35};
        Sort sort = new Sort();
        sort.quicksort(ints,0,6);
        System.out.println(Arrays.toString(ints));

    }
}
class Sort{
    public void quicksort(int[] arr,int left,int right){
        int l = left;
        int r = right;
        int mid = arr[(left + right) / 2];
        while (l < r){
            while (arr[l] < mid){
                l++;
            }
            while (arr[r] > mid){
                r--;
            }
            if (l >= r){
                break;
            }
            int temp = arr[l];
            arr[l] = arr[r];
            arr[r] = temp;
            if (arr[l] == mid){
                r--;
            }
            if (arr[r] == mid){
                l++;
            }
        }

        if (l == r){
            l++;
            r--;
        }
        if (left < r){
            quicksort(arr,left,r);
        }
        if (right > l){
            quicksort(arr,l,right);
        }

    }
}
